# Manim Learning Repository

A collection of Manim code examples and experiments for learners who want to understand mathematical animations and visualizations.

## About

This repository contains Manim animation code for learning and educational purposes. Feel free to explore the code, modify it, and use it as a reference for your own projects.

## Requirements

* Python 3.10+
* Manim Community Edition
* NumPy

## Running the Code

Render a scene with:

```bash
manim -pqh main.py SceneName
```

Replace `SceneName` with the name of the scene class you want to render.

Example:

```bash
manim -pqh main.py ExampleScene
```
